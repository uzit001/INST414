# AI in Medical Imaging: Wikipedia Network Analysis

This project analyzes how AI in medical imaging is structurally positioned within Wikipedia’s hyperlink network.

## Method

- Scraped first-level Wikipedia links
- Built directed graph (94 nodes, 909 edges)
- Removed metadata pages
- Computed:
  - In-degree centrality
  - Betweenness centrality
  - PageRank

## Key Findings

- AI methods form a dense structural core
- Medical imaging acts as a bridge
- Regulatory institutions carry structural influence

## Tools Used

- Python
- requests
- BeautifulSoup
- NetworkX
- matplotlib
